# Docs | AGENTS.md files

`AGENTS.md` files are lean **orientation**, not documentation. Target well under 50 lines.

If you can't summarize why a folder exists in a few words, something is wrong with the folder.

## Shape

Per file:

1. One-line "what is this".
2. A `Structure (/src)` map — one line per folder with a terse, few-word "why it exists". Let the
   agent discover internals; don't narrate nested pieces.
3. `Commands`.
